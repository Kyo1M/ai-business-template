# プロジェクトキックオフ チェックリスト

## 概要

プロジェクト開始時に確認すべき項目のチェックリストです。

---

## 1. プロジェクト基盤

### リポジトリ設定

- [ ] リポジトリを作成した
- [ ] READMEを更新した
- [ ] CLAUDE.mdを更新した
- [ ] .gitignoreを設定した
- [ ] ブランチ保護を設定した（必要に応じて）

### ドキュメント準備

- [ ] docs/フォルダ構成を確認した
- [ ] 初期ドキュメントを作成した
- [ ] メタ情報（title, created, status）を記入した

---

## 2. Discovery（発見）フェーズ準備

### 仮説

- [ ] [hypothesis.md](../docs/01_discovery/hypothesis.md) に初期仮説を記入した
  - [ ] ターゲット仮説（誰が顧客か）
  - [ ] 課題仮説（何に困っているか）
  - [ ] 解決策仮説（どう解決するか）
  - [ ] 価格仮説（いくら払うか）

### ターゲット

- [ ] [target-persona.md](../docs/01_discovery/target-persona.md) にターゲットを記入した
  - [ ] プライマリターゲットを定義した
  - [ ] ペルソナを1人以上作成した
  - [ ] ターゲット外も明確にした

### 課題

- [ ] [problem-statement.md](../docs/01_discovery/problem-statement.md) に課題を記入した
  - [ ] 課題ステートメントを作成した
  - [ ] 課題の優先順位をつけた

### 競合

- [ ] [competitive-analysis.md](../docs/01_discovery/competitive-analysis.md) を記入した
  - [ ] 直接競合をリストアップした
  - [ ] 間接競合・代替手段を把握した

---

## 3. Strategy（戦略）フェーズ準備

### バリュープロポジション

- [ ] [value-proposition.md](../docs/02_strategy/value-proposition.md) を記入した
  - [ ] エレベーターピッチを作成した
  - [ ] 差別化ポイントを明確にした

### ビジネスモデル

- [ ] [business-model.md](../docs/02_strategy/business-model.md) の方向性を記入した

### MVP定義

- [ ] [mvp-definition.md](../docs/02_strategy/mvp-definition.md) の概要を記入した
  - [ ] MVPで検証したい仮説を決めた
  - [ ] スコープ（含めるもの/含めないもの）を決めた

### ロードマップ

- [ ] [roadmap.md](../docs/02_strategy/roadmap.md) を記入した
  - [ ] フェーズを定義した
  - [ ] マイルストーンを設定した
  - [ ] 直近のTODOをリストアップした

---

## 4. 開発準備（必要に応じて）

- [ ] [tech-stack.md](../docs/03_development/tech-stack.md) に技術選定の方向性を記入した
- [ ] 開発環境を準備した
- [ ] 外部サービスのアカウントを作成した

---

## 5. 運用準備

### コミュニケーション

- [ ] 定期的なミーティングを設定した
- [ ] コミュニケーションチャネルを決めた（Slack等）
- [ ] 進捗報告の方法を決めた

### 計測

- [ ] [metrics.md](../docs/04_growth/metrics.md) に初期KPIを設定した
- [ ] 計測ツールを選定した

---

## 6. 最終確認

- [ ] 全ての関係者に共有した
- [ ] 質問・不明点を解消した
- [ ] キックオフミーティングを実施した
- [ ] 最初のタスクを開始した
