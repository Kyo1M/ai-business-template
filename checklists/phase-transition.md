# フェーズ移行チェックリスト

## 概要

各フェーズを完了し、次のフェーズに移行する際に確認すべき項目です。
すべてにチェックが入らなくても移行は可能ですが、重要な項目（⭐）は必ず確認してください。

---

## Discovery → Strategy への移行

### 仮説の検証状況

- [ ] ⭐ ターゲット仮説（H1）を検証した
- [ ] ⭐ 課題仮説（H2）を検証した
- [ ] 解決策仮説（H3）の方向性を確認した
- [ ] 検証結果を `hypothesis.md` に記録した

### ターゲット・課題の明確化

- [ ] ⭐ ターゲット顧客を特定した（`target-persona.md`）
- [ ] ⭐ 解決すべき課題を明確にした（`problem-statement.md`）
- [ ] 5人以上のターゲットと話した
- [ ] 課題の優先順位をつけた

### 競合調査

- [ ] 主要な競合を把握した（`competitive-analysis.md`）
- [ ] 代替手段・間接競合も把握した
- [ ] 差別化の方向性を検討した

### 移行判断

- [ ] ⭐ 「誰の」「何を」解決するか説明できる
- [ ] ⭐ 市場機会があると判断した
- [ ] 学びを `learnings.md` に記録した

**移行日**: ____年__月__日

## Strategy → Development への移行

### 戦略の確定

- [ ] ⭐ バリュープロポジションを定義した（`value-proposition.md`）
- [ ] ⭐ ビジネスモデルを設計した（`business-model.md`）
- [ ] エレベーターピッチが30秒で説明できる

### MVP定義

- [ ] ⭐ MVPで検証する仮説を決めた（`mvp-definition.md`）
- [ ] ⭐ スコープ（やること/やらないこと）を決めた
- [ ] 成功基準（KPI）を設定した
- [ ] 最小限の機能セットを特定した

### ロードマップ

- [ ] ⭐ フェーズ・マイルストーンを設定した（`roadmap.md`）
- [ ] リリース目標日を決めた
- [ ] リソース（人・金・時間）を確保した

### 開発準備

- [ ] 技術スタックを選定した（`tech-stack.md`）
- [ ] 開発環境を準備した
- [ ] 外部サービス・APIのアカウントを準備した

### 移行判断

- [ ] ⭐ 「なぜ作るか」「何を作るか」を説明できる
- [ ] ⭐ MVPで検証したいことが明確
- [ ] ステークホルダーの合意を得た

**移行日**: ____年__月__日

---

## Development → Growth への移行

### MVP完成

- [ ] ⭐ MVPが動作する状態になった
- [ ] ⭐ 最低限の品質を確保した
- [ ] クリティカルなバグがない
- [ ] 基本的なエラーハンドリングができている

### リリース準備

- [ ] ⭐ 計測の仕組みを導入した（`metrics.md`）
- [ ] ユーザー獲得チャネルを準備した
- [ ] サポート体制を整えた
- [ ] リリース告知の準備をした

### ドキュメント整備

- [ ] 要件定義を更新した（`requirements.md`）
- [ ] 設計仕様を更新した（`design-spec.md`）
- [ ] 既知の制限事項を記載した

### 移行判断

- [ ] ⭐ ユーザーに提供できる状態
- [ ] ⭐ フィードバックを収集する準備ができている
- [ ] 改善サイクルを回す体制がある

**移行日**: ____年__月__日

---

## Growth フェーズ内のマイルストーン

### 初期ユーザー獲得（〜100人）

- [ ] 最初の10人のユーザーを獲得した
- [ ] ユーザーインタビューを実施した
- [ ] 致命的な問題を修正した
- [ ] PMF（Product-Market Fit）の兆候を確認した

### 成長軌道に乗せる（100人〜）

- [ ] 主要KPIが改善傾向にある
- [ ] リテンション率が目標を達成した
- [ ] 成長チャネルを特定した
- [ ] 収益化の見込みが立った

---

## ピボット検討のトリガー

以下の状況が発生したら `checklists/pivot-decision.md` を確認：

- [ ] 3ヶ月以上KPIが改善しない
- [ ] ユーザーからのフィードバックが想定と大きく異なる
- [ ] 競合環境が大きく変化した
- [ ] チームの士気が著しく低下した
