# MVPリリース前 チェックリスト

## 概要

MVPをリリースする前に確認すべき項目のチェックリストです。

---

## 1. プロダクト準備

### 機能

- [ ] Must Have機能がすべて実装されている
- [ ] クリティカルなバグが修正されている
- [ ] 主要なユーザーフローが動作する
- [ ] エラーハンドリングが適切に行われている

### UI/UX

- [ ] 主要画面のデザインが完成している
- [ ] レスポンシブ対応している（必要な場合）
- [ ] ローディング状態が表示される
- [ ] エラー時の表示が適切

### パフォーマンス

- [ ] ページ読み込み時間が許容範囲内
- [ ] 重い処理がブロックしない
- [ ] 大量データでも動作する（必要な場合）

---

## 2. 技術的準備

### インフラ

- [ ] 本番環境が構築されている
- [ ] ドメインが設定されている
- [ ] SSL証明書が設定されている
- [ ] 環境変数が正しく設定されている

### セキュリティ

- [ ] 認証・認可が正しく動作する
- [ ] 機密情報がログに出力されない
- [ ] SQLインジェクション対策済み
- [ ] XSS対策済み
- [ ] CSRF対策済み

### データ

- [ ] データベースのバックアップ設定済み
- [ ] 本番データの移行が完了している（必要な場合）
- [ ] 初期データが投入されている

### モニタリング

- [ ] エラー監視ツールが設定されている（Sentry等）
- [ ] ログが収集されている
- [ ] アラートが設定されている

---

## 3. ドキュメント

### ユーザー向け

- [ ] 利用規約が作成されている
- [ ] プライバシーポリシーが作成されている
- [ ] 使い方ガイドがある（必要な場合）
- [ ] FAQがある（必要な場合）

### 運用向け

- [ ] デプロイ手順が文書化されている
- [ ] トラブルシューティングガイドがある
- [ ] 問い合わせ対応フローが決まっている

---

## 4. 計測準備

### アナリティクス

- [ ] Google Analytics（または同等）が設定されている
- [ ] イベントトラッキングが設定されている
- [ ] コンバージョン計測が設定されている

### KPI

- [ ] 主要KPIが定義されている
- [ ] 計測方法が決まっている
- [ ] ダッシュボードが準備されている

---

## 5. マーケティング準備

### ランディングページ

- [ ] LPが完成している
- [ ] OGP画像が設定されている
- [ ] SNSシェア時の表示を確認した

### 告知準備

- [ ] 告知文が作成されている
- [ ] 画像・動画素材が準備されている
- [ ] 公開日時が決まっている

---

## 6. サポート準備

### 問い合わせ対応

- [ ] 問い合わせフォームがある
- [ ] 問い合わせ先メールアドレスが設定されている
- [ ] 対応フローが決まっている
- [ ] 対応時間が決まっている

### 緊急時対応

- [ ] 緊急連絡先が共有されている
- [ ] ロールバック手順が準備されている
- [ ] 緊急時の判断基準が決まっている

---

## 7. 最終確認

### テスト

- [ ] 本番環境で動作確認した
- [ ] 決済フローをテストした（決済がある場合）
- [ ] メール送信をテストした（メールがある場合）
- [ ] 異なるデバイス・ブラウザで確認した

### レビュー

- [ ] 関係者に最終確認を依頼した
- [ ] フィードバックを反映した

### Go/No-Go判断

- [ ] リリース基準を満たしている
- [ ] リリース日時が確定した
- [ ] 関係者に周知した
