# MVP仕様生成プロンプト

## 使用方法

- MVP戦略策定後、具体的な仕様に落とし込む際
- 開発着手前の要件整理に

## プロンプト本文

```

以下の情報をもとに、MVP仕様書を作成してください。
```

## 使用例

### 入力例

```
## 事業概要
フリーランス向けの請求書作成・管理サービス

## ターゲット顧客
副業を始めたばかりのエンジニア（請求書作成が初めて）

## 解決したい課題
- 請求書の作り方がわからない
- Excelで作るのが面倒
- 入金管理が煩雑

## 優先機能
- 請求書のテンプレートから簡単作成
- PDFダウンロード
- 入金ステータス管理

## 制約条件
- 開発期間: 2週間
- 予算: 30万円
- 技術的制約: 個人開発（1人）
```
