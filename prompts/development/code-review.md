## コード
```javascript
async function getUser(id) {
  const res = await fetch('/api/users/' + id)
  const data = await res.json()
  return data
}

async function updateUser(id, name, email) {
  const res = await fetch('/api/users/' + id, {
    method: 'PUT',
    body: JSON.stringify({name: name, email: email})
  })
  return res.json()
}
```

## コンテキスト
