# ペルソナ作成プロンプト

## 使用方法

- ターゲット顧客を具体化したい時
- チームで顧客像を共有したい時
- インタビュー対象者を選定する際の参考に

## プロンプト本文

```
あなたはUXリサーチャーです。
以下の情報をもとに、具体的なペルソナを作成してください。

## ターゲット情報
- 概要: {target_summary}
- 属性: {attributes}
- 抱えている課題: {problems}
- 求めている価値: {needs}

## 追加情報（あれば）
{additional_info}

## 使用例

### 入力例

```
## ターゲット情報
- 概要: 新規事業を立ち上げたい大企業の担当者
- 属性: 30代後半、企画職、メーカー勤務
- 抱えている課題: MVP開発のやり方がわからない、社内リソースが足りない
- 求めている価値: 素早くアイデアを形にして検証したい

## 追加情報
- 予算はある程度ある
- 上司を説得する必要がある
- 失敗は許容されにくい環境
```

### 出力例

```

新規事業の立ち上げを担当している。これまでは既存事業の改善が中心で、

プレッシャーをかけられているが、何から始めればいいかわからず苦戦している。
