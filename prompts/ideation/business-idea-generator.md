# ビジネスアイデア整理プロンプト

## 使用方法

- アイデアの初期段階で使用
- 頭の中のアイデアを言語化したい時
- チームで認識を揃えたい時

## プロンプト本文

```
あなたはスタートアップのアドバイザーです。
以下のビジネスアイデアを構造化して整理してください。

## アイデアの概要
{idea_description}

## 追加情報（あれば）
- 想定ターゲット: {target}
- きっかけ・背景: {background}
- 解決したい課題: {problem}

---

以下の形式で整理してください：

### 1. エレベーターピッチ（30秒で説明）
- 誰向けの
- 何を提供する
- どんなサービス/プロダクトか

### 2. ターゲット顧客
- 誰が顧客か（具体的に）
- なぜその人たちか

### 3. 課題
- 顧客はどんな課題を持っているか
- その課題はどれくらい深刻か
- 今はどう対処しているか

### 4. 解決策
- どうやって課題を解決するか
- なぜこの解決策が有効か

### 5. 差別化ポイント
- 競合や代替手段との違い
- 選ばれる理由

### 6. 検証すべき仮説
- このアイデアで最もリスクの高い仮説は何か
- どうやって検証するか

### 7. 質問・確認事項
- さらに明確にすべき点
- 追加で聞きたいこと
```

## 使用例

### 入力例

```
## アイデアの概要
フリーランスエンジニア向けの案件マッチングサービス。
AIがスキルや希望条件を分析して、最適な案件をレコメンドする。

## 追加情報
- 想定ターゲット: 副業で稼ぎたいエンジニア
- きっかけ・背景: 自分が案件探しに苦労した経験から
- 解決したい課題: 良い案件を見つけるのに時間がかかる
```

### 出力例

```
### 1. エレベーターピッチ
副業で稼ぎたいエンジニア向けの、AIによる案件マッチングサービスです。
スキルや希望条件を登録するだけで、最適な案件が自動でレコメンドされ、
案件探しの時間を大幅に削減できます。

### 2. ターゲット顧客
- **誰が**: 副業で収入を増やしたい現役エンジニア（25-40歳）
- **なぜ**: 
  - 本業があり案件探しに使える時間が限られている
  - スキルはあるが最適な案件を見つけるのに苦労している

### 3. 課題
- **課題**: 良い案件を見つけるのに多大な時間がかかる
- **深刻度**: 中〜高（時間は限られた資源であり、機会損失が発生）
- **現在の対処法**: 
  - 複数のマッチングサイトを巡回
  - 知人からの紹介を待つ
  - SNSで情報収集

（以下略）
```
