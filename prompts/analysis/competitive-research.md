# 競合調査プロンプト

## 使用方法

- 新規事業の競合を調査する時
- 差別化ポイントを明確にしたい時
- 市場を理解したい時

## プロンプト本文

```
あなたは市場調査のアナリストです。
以下の情報をもとに、競合分析を行ってください。

{service_description}

## 競合候補（わかる範囲で）
{competitors}

## 分析の目的
{purpose}
```

## 使用例

### 入力例

```
## 自社サービス概要
AIを活用したMVP開発支援サービス。
アイデアからMVPリリースまでを2週間で実現する。
主に新規事業担当者やスタートアップがターゲット。

## 競合候補
- 従来の開発会社
- ノーコードツール（Bubble等）
- MVP開発サービス（数社名前を知っている程度）

## 分析の目的
差別化ポイントを明確にして、営業時のトークに活かしたい
```

## 注意事項

- このプロンプトの結果は一次情報ではありません
- 重要な情報は必ず公式サイト等で確認してください
- 最新の情報は反映されていない可能性があります
