## 実験のベストプラクティス

### 実験前チェックリスト

- [ ] 仮説が明確に定義されている
- [ ] 成功基準が数値で定義されている
- [ ] 十分なサンプルサイズが確保できる
- [ ] 実験期間が適切に設定されている
- [ ] 計測の仕組みが準備されている

### 実験中の注意点

- 途中で設定を変更しない
- 複数の変更を同時にテストしない
- 外部要因の影響を考慮する

### 実験後のアクション

- [ ] 結果を記録する
- [ ] 統計的有意性を確認する
- [ ] 学びを言語化する
- [ ] ネクストアクションを決める
- [ ] 成功した場合は本番適用する
