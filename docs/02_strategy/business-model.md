## ユニットエコノミクス

### 主要指標

| 指標 | 値 | 目標 |
|------|-----|------|
| **CAC**（顧客獲得コスト） | | |
| **LTV**（顧客生涯価値） | | |
| **LTV/CAC** | | 3以上が目安 |
| **Payback Period**（回収期間） | | ヶ月 |
| **Churn Rate**（解約率） | | %/月 |

### 計算式

```
LTV = 月額単価 × 平均継続月数 × 粗利率
CAC = マーケティング費用 ÷ 新規顧客数
```
