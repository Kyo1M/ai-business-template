# バリュープロポジション

## 概要

顧客に提供する価値を明確に定義します。バリュープロポジションキャンバスを使って整理します。

---

## エレベーターピッチ

30秒で説明できる自社サービスの説明

```
【ターゲット顧客】向けの【サービス名】は、
【カテゴリ/種類】です。

【主要なベネフィット】を提供し、
【競合や代替手段】とは違って、
【独自の差別化ポイント】が特徴です。
```
