## 使い方

### 一覧（CSV）と詳細（プロファイル）の役割分担

- **`competitive-analysis.md`（サマリー）**: 競合分析の結論と意思決定に必要な情報
  - 目的: ランドスケープ把握、類型化、ポジショニング決定
  - 粒度: 比較表、差別化ポイント、戦略的示唆
- **個別プロファイル（`.md`）**: 「狭く深く」差別化の根拠を取るための詳細
  - 目的: 価格体系、訴求、導入事例、提供プロセス、強み/弱みの裏取り
  - 粒度: サイト/IR/記事/登壇などの一次情報リンクつきで記録

### 更新フロー（推奨）

1. まず `competitive-analysis.md` で全体像を把握
2. 深掘りが必要な競合は `templates/competitor-profile.md` をコピーして個別プロファイルを作成
3. 収集した資料（PDF等）は `assets/competitors/` に保存
4. サマリー（意思決定に効く結論）は `docs/01_discovery/competitive-analysis.md` に反映

### 命名ルール（個別プロファイル）

- ファイル名は `{会社名またはサービス名}.md`（小文字、ハイフン区切り）
- 例: `techstar.md`, `innovate-inc.md`
