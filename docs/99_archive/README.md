# 📦 アーカイブ

役目を終えた、または古くなったドキュメントを保管するフォルダです。

## 用途

- 仮説が大きく変わった際の旧仮説
- ピボット前の戦略ドキュメント
- 廃止された機能の仕様書
- 過去のロードマップ
- その他、参照価値はあるが現行ではないドキュメント

## アーカイブのルール

### いつアーカイブするか

- ドキュメントの内容が現状と大きく乖離したとき
- ピボットや方針転換があったとき
- 新しいバージョンのドキュメントに置き換えられたとき

### アーカイブ方法

1. 元のフォルダからこのフォルダに移動
2. ファイル名の先頭に日付を追加

```
# 移動前
docs/01_discovery/hypothesis.md

# 移動後
docs/99_archive/2025-01-15-hypothesis-v1.md
```

3. 必要に応じてファイル内にアーカイブ理由を追記

```markdown
---
archived: 2025-01-15
archive_reason: ピボットに伴い仮説を刷新したため
---
```

### 参照リンクの更新

元のドキュメントへのリンクがある場合は、新しいドキュメントへのリンクに更新してください。

## 注意事項

- **削除ではなくアーカイブ**: 過去の意思決定の経緯を残すため、削除せずアーカイブ
- **定期的な整理**: 半年以上経過したアーカイブは、本当に必要か見直す
- **機密情報に注意**: アーカイブ前に機密情報が含まれていないか確認
